# go-rcon

**go-rcon** is a Go library for querying Source RCON servers.

## Requirements

* Go 1.1 or above

## Installation

```
go get github.com/galexrt/go-rcon
```

## Update

```
go get -u github.com/galexrt/go-rcon
```
